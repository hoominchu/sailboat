What is Sailboat?

Today's web browsers have failed in some key areas of web browsing such as tab clutter, tab management, lack of support for preserving user privacy, limited value in browser history and lack of ease of use in using bookmarks etc. While tabs make browsing much convenient they also add to visual clutter and distract users away from the task at hand. Users of web browsers today swim in a sea of haphazardly organized tabs, bookmarks and downloads. This is because browsers operate at the level of individual web pages, and rarely understand the user’s high-level tasks. Sailboat addresses this problem by compartmentalizing the web browsing experience across different tasks defined by the user.

In Sailboat, different tasks have their own tabs, history and bookmarks while maintaining a unified identity across compartments.

With Sailboat, you can:

  • You can create and switch easily between tasks such as ”Philosophy Paper”, ”Summer Internships”, and ”Trip to China” letting you manage browser tabs and windows more efficiently.
  • Keep your tabs, windows, and bookmarks organized.
  • You can archive tasks that you are done with while preserving all the browsing data and revisit it months later. Soon you will be able to share a task with someone too!
  • See the time spent on different tasks to aid productivity.
These features provide the foundation for many more novel features that we would like to develop in the future.

This extension is currently in beta. If you come across any bugs (and most likely you will!), have some cool features you'd like to see in Sailboat or if you have any feedback, drop me an email at hoominchu@gmail.com.


Features


Task based browsing

Simply open a new tab to see all your tasks. You can create, open, rename, archive or delete a task on this page. You can also check the time spent on each task at the bottom of the task card. Archive a task once you are done with it.


Dock

A dock is shown on every webpage you open. This gives you quick access to all your current tasks.

Quick tip: If the dock is hiding/blocking the content on the page, press 'control' key to hide the dock temporarily.
