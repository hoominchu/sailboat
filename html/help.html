Quick Guide

Tasks

Sandbox your browsing tasks— tabs, bookmarks, archived pages, history, downloads folder.

This helps in reducing tab clutter as you can have a lot more tabs open at a time as they are segregated into different tasks. You don’t have to close and lose tabs.

Provides a completely different contexts for different Tasks with different bookmarks and archived pages etc.

A new folder is created for every Task in the Downloads folder and all your downloads are saved there making it much easier to manage your Downloads folder. Goodbye messy Downloads folder!

If you are not working on any particular task and want to do some general browsing switch to 'Default' task.

Task based analytics on history are coming soon!

Dock

The Sailboat dock helps you work with Tasks in an easy and quick way.

There will be some common Tasks in your dock right when you install it. You can continue to use them or delete them.

Click on any Task button in the Dock to switch to that Task. Alternatively you can use shortcut ‘Control+~’ to switch between Tasks.

Create a new task — Click on the ‘+’ button at the extreme right on the dock. Press ‘enter’ after entering the name of Task.

Press ‘+’ beside the Task button to add the current tab to that Task.

Click on ‘X’ to close that Task.

Hover on a Task button to see the Tabs open in that Task.

Click on the ‘Archive’ button to archive the current page.

Collapse/Expand Dock button is to the extreme left of the Dock.

Hover on any link on the page to see if a link is already open in one of the tabs. This helps in preventing the same URL multiple times.

Archived Pages

When you archive a page it gives you the ability to search through the full content of the page.

We think it will be quite useful in re-finding information.

So if you find yourself in a position when you sort of remember reading some word somewhere use this feature!

Oh yes don’t forget to archive pages before you try to use this!
